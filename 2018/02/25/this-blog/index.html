What is it all about?

Published Sun, Feb 25, 2018 by Danny

What I’ve learned in my time as a Ph.D. student is that I enjoy is facilitating other people’s work and research. I love writing R code that helps other people succeed, be it code that’s necessary to get something working for a project of my adviser’s, the code to build a function or a shiny app to make statistical methods available to researchers, or code to help someone with understanding, transforming, or analyzing their data. So my hope is that after (or maybe while) I write my dissertation, I’ll be able to find work in consulting methodology, data science, or any sort of statistical analysis. I’ll use this page to show off snippets of work or small discoveries I’ve made while I work with other researchers or write my own code.
